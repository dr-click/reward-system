# README

## Reward System

System for calculating rewards based on recommendations of customers.


## Start Project:

  > run `bundle install`

  > run `ruby main.rb`


## Project Details:

* Environment

  > Ruby 2.6.1

  > Rspec

## API Cals:

  > POST /rewards : Calculate the rewards score.

    > Data attributes : file (contains the recommendations list).

## Tests

  > run test `rspec`
